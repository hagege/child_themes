# Changelog - GSG Gymnasium Child Theme

## Version 1.0.0 (2026-03-23)

### Hinzugefügt
- Initiales Release
- Vollständiges theme.json Design System
- 8 Block Patterns in Kategorie "GSG"
- Custom Post Types (Termine, Fachbereiche)
- Header und Footer Template Parts
- Home Template
- Inter Schriftart mit Google Fonts
- Responsive Design
- Accessibility Features (WCAG 2.1)
- Performance-Optimierungen

### Schriftarten
- **Primary:** Inter (Google Fonts) mit System-Fallback
- **Fallback Stack:** -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif

### Patterns
Alle Patterns sind in der Kategorie **"GSG"** verfügbar:
1. GSG Hero Section
2. GSG News Grid
3. GSG Fachbereiche
4. GSG Termine Liste
5. GSG Anmeldung CTA
6. GSG Info Box
7. GSG Accent Box
8. GSG Statistiken

### Theme Features
- theme.json mit 11 Farben aus GSG Corporate Design
- 4 Gradients (Hero, Section, Yellow, Overlay)
- 8 Font Sizes (responsive)
- 9 Spacing Presets (4px - 96px)
- 6 Shadow Presets
- Custom Properties für Transitions, Border-Radius, Z-Index

### Technische Details
- **WordPress:** 6.4+
- **PHP:** 7.4+
- **Parent Theme:** Ollie
- **Text Domain:** gsg-child
