## 📊 ÄNDERUNGSDETAILS

### 1. **Neue Hilfsfunktionen (Abwärtskompatibilität)**

**NEU: Zentrale Konfiguration**

```php
/**
 * Get all plugin option defaults
 * @since 1.6
 */
function slsh_get_option_defaults(): array {
    return array(
        'slsh_header_shrink_height'           => 80,
        'slsh_animation_duration'             => 0.8,  // Geändert: 0.6 → 0.8
        'slsh_heigth_header'                  => 100,  // Geändert: 120 → 100
        // ... weitere Defaults
    );
}

/**
 * Get single option with default value
 * @since 1.6
 */
function slsh_get_option( string $option_name, $default = null ) {
    $defaults = slsh_get_option_defaults();
    if ( null === $default && isset( $defaults[ $option_name ] ) ) {
        $default = $defaults[ $option_name ];
    }
    return get_option( $option_name, $default );
}
```

**Vorteil:**
- ✅ Zentrale Stelle für alle Default-Werte
- ✅ Einfacher zu testen und zu debuggen
- ✅ Macht Code wartbarer

**Kompatibilität:**
- ✅ Alte Werte werden respektiert (via `get_option()`)
- ✅ Nur neue Installationen nutzen neue Defaults

### 3. **Verbesserte Sanitierung (Sicherheit)**

**VOR (v1.5):**
```php
register_setting(
    'slsh_options_group',
    'slsh_animation_duration',
    array(
        'type'              => 'float',
        'sanitize_callback' => 'floatval', // Keine Grenzen!
    )
);
```

**NACH (v1.6):**
```php
register_setting(
    'slsh_options_group',
    'slsh_animation_duration',
    array(
        'type'              => 'number',
        'sanitize_callback' => function( $value ) {
            $float_val = (float) $value;
            // Clamp zwischen 0.1 und 5 Sekunden
            return max( 0.1, min( 5, $float_val ) );
        },
        'default'           => 0.8,
    )
);
```

**Vorteil:**
- ✅ Keine ungültigen Werte (z.B. -100s oder 999s)
- ✅ Plausible Grenzen erzwungen
- ✅ Sicherer

**Kompatibilität:**
- ✅ Alte Werte werden normalisiert (nicht gelöscht)
- ✅ Nutzer merkt nichts

### 8. **Plugin Activation/Deactivation Hooks**

**NEU in v1.6:**

```php
function slsh_plugin_activation(): void {
    // Initialize default options on activation
    $defaults = slsh_get_option_defaults();
    foreach ( $defaults as $option_key => $default_value ) {
        if ( false === get_option( $option_key ) ) {
            add_option( $option_key, $default_value );
        }
    }
    flush_rewrite_rules();
}
register_activation_hook( __FILE__, 'slsh_plugin_activation' );

function slsh_plugin_deactivation(): void {
    // Optional: delete options on deactivation
    flush_rewrite_rules();
}
register_deactivation_hook( __FILE__, 'slsh_plugin_deactivation' );
```

**Vorteil:**
- ✅ Saubere Aktivierung/Deaktivierung
- ✅ Optionales Cleanup
- ✅ Flush Rewrite Rules
